Разработка gRPC-сервиса для управления задачами, (task manager)
Требования к заданию:
Функциональность gRPC-сервиса:

CreateTask добавляет новую задачу
ListTasks возвращает список всех задач
GetTask возвращает задачу по ее id
UpdateTask обновляет существующую задачу
DeleteTask удаляет задачу по ее id
Архитектура проекта: Чистая архитектура
Технологии: Go, gRPC, PostgreSQL, GORM, Docker, GitHub
Дополнительные требования: Документация, Чистота кода, Обработка ошибок, Валидация данных.
